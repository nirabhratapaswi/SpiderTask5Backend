== README

Install Rails 4 (comes with inbuilt installation of Ruby 1.9 and git)

Clone this repository to "C:\Sites" (or elsewhere if you wish)

git bash and type:
    $ cd "C:\Sites\spider5"
    $ bundle install
    $ bin/rails server
    
Then go to "http:\\localhost:3000" or simply "localhost:3000" and start using omdb api smoothly!!

..enjoy
